💻 Sobre o Projeto

Esta é uma landing page responsiva, desenvolvida utilizando HTML, CSS3 e JavaScript, com foco em:

Organização de código

Experiência do usuário (UX)

Boas práticas de desenvolvimento front-end

Conversão de visitantes em contatos

O principal objetivo do projeto é apresentar a banda de forma clara, moderna e estratégica, incentivando o usuário a entrar em contato para contratação.

🚀 Funcionalidades

✅ Estrutura clean e visual profissional

✅ Responsividade mobile (menu adaptável, grid e cards)

✅ Interações dinâmicas com JavaScript

✅ Formulário com integração direta ao WhatsApp

✅ Agenda de eventos dinâmica com botão "ver mais / ver menos"

✅ Navegação com destaque ativo ao scroll

✅ Seção de redes sociais

✅ Players incorporados (YouTube e Spotify)

🛠️ Tecnologias Utilizadas

HTML5

CSS3

JavaScript

📱 Responsividade

O projeto foi desenvolvido com abordagem responsiva, garantindo boa experiência tanto em dispositivos móveis quanto em desktops.
